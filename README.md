# Chat Websocket

Aplicação Chat Websocket construído com a linguagem PHP e utilizando o framework Ratchet

## Pré Requisitos
- PHP
- [Composer](https://getcomposer.org/download/)

## Instalação
Após fazer o clone do repositório na sua máquina digite o seguinte comando no terminal:

```bash
composer install
```

Em seguida, inicie o servidor websocket com o arquivo server.php com o seguinte comando no terminal:

```bash
php server.php
```

Após isso, abra o arquivo **index.php** e faça login com seu usuário para acessar o chat.
